# 家計簿アプリ

個人の収支を記録・管理できる Web アプリケーションです。

## 🌐 デモ

https://kishiroharuka.github.io/kakeibo-app/

## 📱 機能

- ユーザー認証（新規登録・ログイン）
- 収入・支出の記録
- 月別・年別の集計
- グラフによる可視化
- 平均支出の計算
- レスポンシブデザイン（スマホ対応）

## 🛠️ 使用技術

- HTML / CSS / JavaScript
- Firebase Authentication（認証）
- Firebase Firestore（データベース）
- Chart.js（グラフ描画）
- GitHub Pages（ホスティング）

## 💡 工夫した点

- 手書き風のデザインで親しみやすいUI
- リアルタイムでデータ同期
- 詳細な月別表示で履歴を確認しやすく
